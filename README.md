# ReScript Project Template

The only official ReScript starter template.

## Installation

```sh
npm install
```

## Build

- Build: `npm run re:build`
- Clean: `npm run re:clean`
- Build & watch: `npm run re:start`

## Run

```sh
node src/Week${W}/Year${YYYY}Day${DD}.bs.js
```
